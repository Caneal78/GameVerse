# GameVerse User Guide

## Getting Started
1. Launch the app.
2. Create a new project or load an existing vault.
3. Choose a folder for the project when prompted.
4. Use the dashboard to create and manage items.

## Core Areas
- Dashboard: browse items by category, status, and tag
- World Bible: store lore, locations, factions, and background notes
- Collections: group related items together
- Item Detail: edit fields, notes, links, files, and versions
- Settings: adjust templates, back up the vault, and set Blender path

## Files
- Import files from the item detail page
- Use copy mode for files that should live inside the vault
- Use link mode only for external files you want referenced in place
- Restore previous versions from the file list when needed

## Backups And Export
- Create backups from the sidebar or settings page
- Export an item when you want a portable bundle with docs and assets

## Desktop Tips
- Keep each project in its own folder
- Avoid moving files inside the vault by hand unless you know the references will be updated
- Update Blender in Settings if the app cannot find it automatically
